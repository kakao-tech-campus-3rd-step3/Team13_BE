## 개요
- 기간: 2025.08 ~  
- 내용: **교내 스포츠 매칭 플랫폼**  
  - 대학생들이 **운동 시간 부족**, **파트너 부재**, **시설 접근성 문제**로 운동 참여가 어려운 문제를 해결하기 위해 제작  
  - 학생들이 **간편하게 운동 파트너를 찾고**, **매칭 후 커뮤니케이션과 피드백**을 통해 지속적인 운동 문화를 형성하도록 지원  

**핵심 기능**
- 회원 관리: 대학 계정 기반 인증, 프로필 및 선호 운동 저장  
- 퀵 매칭: 시간·장소 기반 즉시 매칭  
- MMR 및 랭크 기반 매칭 알고리즘: 사용자 실력과 선호를 고려한 공정 매칭
- 실시간 알림: 매칭 및 경기 일정 푸시 알림  
- 운동 기록/피드백: 경기 후 평가, MVP 선택, 승패 기록  

---

## 기술 스택 & 아키텍처
- **백엔드:** Spring Boot, JWT  
- **프론트엔드:** React
- **데이터베이스:** PostgreSQL  
- **CI/CD:** GitHub Actions, AWS EC2
